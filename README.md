# BeboFix Website

On-Demand Home Services Platform & B2B Marketing Agency

**Domain:** bebofix.com
**Company:** Vola Capital Ltd (Company No. 16888064)

## Pages
- `index.html` — Homepage (hero, services, how it works, B2B marketing, CTA)
- `services.html` — All 14 categories with 100+ subcategories and pricing
- `for-professionals.html` — For tradespeople (benefits, 3 pricing tiers)
- `marketing.html` — B2B Marketing services (Starter/Growth/Enterprise packages)
- `contact.html` — Contact form + company details

## Tech Stack
- Pure HTML/CSS (no frameworks, no build step)
- Google Fonts: DM Sans + Playfair Display
- Mobile responsive
- Self-contained pages (CSS inlined)

## Deploy
Upload all `.html` files to web root on your server.

## Design System
- **Colors:** Navy `#0F2B3C`, Teal `#1DB8A4`, Blue `#2E86AB`, Orange `#F77F00`
- **Fonts:** Playfair Display (headings), DM Sans (body)
- **Style:** Fresh, modern, service-oriented
